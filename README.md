# Royal Pouch

GitHub Pages / PWA-ready mobile web app.

## Upload to GitHub Pages

Upload all files in this folder to your repository root:

- `index.html`
- `style.css`
- `script.js`
- `manifest.json`
- `sw.js`
- `icons/`

Then enable GitHub Pages for the repository.
